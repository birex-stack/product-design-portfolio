# Maciej Forc — Product Design Portfolio

Premium product portfolio demonstrating how complex product problems are solved.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output in `dist/` — deployed automatically via GitHub Actions.

## Structure

- `index.html` — Home (About, Challenges, Principles, Contact)
- `case-studies/` — Dedicated challenge pages (IBM XFTM, Elastic SLO, Elastic Cases, Otomoto)
